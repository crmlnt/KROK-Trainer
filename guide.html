KROK Trainer Guide

How to use the trainer

1. Select a subject

Choose a specific subject or keep “All Subjects”.

2. Start Exam Mode

Choose the number of questions: 10, 20, 50, 100, 200 or All.

3. Review your result

At the end of the exam you will see correct answers, wrong answers and accuracy.

4. Export Error Log

If you make mistakes, you can export them and paste them into your personal study log.

5. Export Exam Log

The Export Exam Log button allows you to download a CSV file containing all the questions answered during the current Exam Mode session.

Unlike the Error Log, which saves only wrong answers, the Exam Log saves both correct and wrong answers. This is useful for a more complete analysis of your performance.

  • Date: date of the exam session
  • Subject: subject of the question
  • Question: full question text
  • Your Answer: the answer you selected
  • Correct Answer: the correct answer
  • Result: Correct or Wrong

You can upload this CSV file to ChatGPT to receive a more accurate study analysis, because it shows not only your mistakes, but also the questions you answered correctly.

Important: the Exam Log is created only during Exam Mode. Practice Mode does not generate an Exam Log.

6. Review Mistakes

This mode lets you repeat only the questions you answered incorrectly.

7. Statistics

The statistics dashboard shows your performance by subject and highlights weak areas.
